SQL SERVER 2016 Features

Database Engine:

Multiple tempdb's can be setup during installation
JSON support is now available
Temporal tables are history tables which store data changes
Polybase query engine integrates external data in HDFS or Azure blob storage
New Query store stores query text, execution plans and performance issues
Stretch database feature lets you dynamically, securly archive data from local sql server db to Azure Db
In=memory OLTP supports tables upto 2 tb

Direct query can query live external data
